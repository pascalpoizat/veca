Obtaining VECA

In order to get VECA you will need:

  • the VECA plugin for Eclipse
    to edit models in the VECA DSL and transform them into the VECA JSON format
  • the VECA to timed automata transformer
    to transform models in the VECA JSON format into timed automata in the XTA format

In order to perform timed verification of the models once transformed in timed automata, you will need to install one of these external tools:

  • ITS Tools
  • UPPAAL
